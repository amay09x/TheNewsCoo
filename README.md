# TheNewsCoo 📰

TheNewsCoo is a desktop AI application that:
• Summarizes news articles
• Translates summaries into multiple languages
• Converts translated text into speech

Built using:
- Python
- CustomTkinter
- Azure Cognitive Services (Text Analytics, Translator, Speech)

## Features
✔ Extractive summarization  
✔ Multi-language translation  
✔ Neural text-to-speech  
✔ Clean desktop UI

## Run Locally

pip install -r requirements.txt  
python main.py
